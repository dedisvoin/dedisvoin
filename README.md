## About Me

My name is Ivan Pavlov, and I am a C++ and Python software developer with a niche focus on the foundational technologies of the digital world: game engines and compilers.

While I enjoy full-stack development, my true passion lies "under the hood." On the game development side, I specialize in building robust, data-oriented game engines in C++ that serve as the foundation for complex interactive worlds. I focus on creating efficient render loops, implementing physics simulations, and designing flexible entity-component architectures.

Parallel to this, my interest in programming languages has led me deep into the world of compiler construction. I have experience working with the full pipeline of language implementation—from lexing and parsing (using tools like Flex/Bison or hand-rolled recursive descent) to semantic analysis and code generation. I believe that understanding how compilers work makes me a better engineer across all domains.

I am always looking for challenging projects that push the limits of hardware and require a deep understanding of computer architecture and system design.
